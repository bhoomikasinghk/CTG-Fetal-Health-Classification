# CTG Fetal Health Classification

## Objective
Classify fetal health into:
- Normal
- Suspect
- Pathologic

## Dataset
CTG (Cardiotocography) Dataset

## Machine Learning Model
- Logistic Regression (One-vs-Rest)

## EDA
- Data Cleaning
- Correlation Heatmap
- Class Distribution Analysis

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
